# Multi-Language Process Terminator

A professional Windows batch utility for selective termination of programming environments.

## Features

- Targeted termination for Python, C#, Java, Javascript (Node.js), and C++.
- Interactive selection menu for granular control.
- Global termination option for all supported environments.
- Native performance with zero external dependencies.

## Usage

1. Execute `ExeKiller.exe`.
2. Select the target environment from the interactive menu (1-6).
3. The utility will forcefully terminate all related processes.
4. Press any key to return to the main menu for further actions.

## Termination Coverage

- **Python**: python.exe, pythonw.exe, py.exe
- **C#**: dotnet.exe
- **Java**: java.exe, javaw.exe
- **Javascript**: node.exe
- **C++**: cl.exe, gcc.exe, g++.exe, make.exe

Warning: All terminations are forceful (/F). Ensure all critical work is saved before execution.

## Security Note

Windows may show a "Publisher could not be verified" warning or a "Windows protected your PC" screen because the executable is not digitally signed. This is normal for custom-built tools.

To run it:
1. Click **"More info"**.
2. Click **"Run anyway"**.
